# DappReader


## 什么是DappReader?

![cover](https://github.com/LidamaoHub/abiViewer/blob/master/img/intro.png)


DappReader是一款方便的可以私有读取智能合约的小工具

适用于dapp开发者在Etherscan之外快速交互智能合约或与团队做内部分享测试使用


## 快速安装

下载本项目到本地后

```
yarn install
```

### 在本地运行
```
yarn run serve
```

### 编译本项目
```
yarn run build
```
